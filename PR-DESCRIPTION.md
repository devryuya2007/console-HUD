# 概要
- ポップアップ内でショートカット説明を見せるための小さい HUD を整理。
- 各操作（ハードリセット・Console 表示・localStorage 表示）を列で並べ、文字サイズと表示を整えた。

# 変更内容
- `App.tsx` を 360px 正方形の枠のまま説明カードを縦に並べるレイアウトに更新。
- Tailwind のクラスで文字の大きさ・太さを上げ、各カードの色合いとシャドウを整えた。
- `data-testid="hud-frame"` を保ちつつ、テストで表示枠の存在を確認。

# 動作確認
- `npm run lint`
- `npm run test`

# 影響範囲 / リスク
